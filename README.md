# redux-saga-beginner
[![npm version](https://img.shields.io/npm/v/redux-saga.svg)](https://www.npmjs.com/package/redux-saga)

# Instructions

Setup

```
// clone the repo
git clone https://github.com/rajanmagar/react-redux-saga.git

cd redux-saga-beginner

npm install || yarn
```

Run the demo

```
npm start || yarn start
```

Run tests

```
npm test
```
